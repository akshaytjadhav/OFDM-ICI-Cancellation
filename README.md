# OFDM Inter Carrier Interference (ICI) Cancellation

## Overview

This repository contains my M.E. Communication Engineering project on reducing Inter Carrier Interference (ICI) in OFDM systems.

## Objectives

- Study OFDM systems
- Analyze impact of ICI
- Implement self-cancellation technique
- Improve system performance

## Tools Used

- MATLAB
- Communication Toolbox
